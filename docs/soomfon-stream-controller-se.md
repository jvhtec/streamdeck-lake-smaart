# Soomfon Stream Controller SE

## Overview
The Soomfon Stream Controller SE is a compact control surface with multiple programmable LCD keys, rotary encoders, and tactile buttons. This document records the physical layout so it can be referenced when mapping actions or creating Stream Deck-style profiles.

## Layout
Observed layout from the Soomfon product imagery:

- **LCD keys:** 6 total, arranged in a 2 × 3 grid.
- **Primary encoder:** 1 large rotary encoder near the upper-right corner.
- **Secondary encoders:** 2 smaller rotary encoders positioned to the right of the LCD keys.
- **Tactile buttons:** 4 round buttons aligned along the front edge.
